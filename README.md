# 팀프로젝트(KIOSK 제작)

### 프로젝트 목적

 메가커피 키오스크를 바탕으로 시중에 있는 키오스크 프로그램 구현.

### 구동시 주의사항
 
💬 DB 테이블 폴더에 있는 데이터 테이블 설정을 로컬호스트 내에 먼저 구성한다. <br>
💬 database.OjdbcConnection 클래스 내의 멤버변수를 본인의 DB 설정에 맞춘다. <br>
💬 Main클래스는 project 패키지 내의 Execution 클래스이다. <br>
